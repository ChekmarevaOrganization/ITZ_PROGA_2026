# ITZ_PROGA_2026
# Формирование сводного файла учебной нагрузки

## Описание проекта
Скрипт автоматически объединяет данные об учебной нагрузке преподавателей
за осенний и весенний семестры в один сводный CSV-файл текущего учебного года.
Все данные переносятся без изменения структуры и содержимого.

Установка библиотеки:
pip install pandas

## Входные файлы
 `autumn.csv` | Данные об учебной нагрузке за осенний семестр 
 `spring.csv` | Данные об учебной нагрузке за весенний семестр

Оба файла должны находиться в той же папке, что и скрипт.
Разделитель полей в файлах — точка с запятой (`;`).

## Структура входных данных
Оба файла содержат одинаковые колонки:
| Колонка | Описание |
|---|---|
| Дисциплина | Название учебной дисциплины |
| Вид нагрузки | Тип занятия (лекция, практика, семинар и т.д.) |
| Группы | Учебные группы, для которых ведётся занятие |
| Нагрузка | Количество часов нагрузки |
| Поток | Признак потокового занятия |
| Количества часов по УП | Часы по учебному плану |
| Табельный № | Табельный номер преподавателя |
| ФИО | Фамилия, имя, отчество преподавателя |
| Должность | Должность преподавателя |
| Желаемая аудитория | Предпочтительная аудитория для проведения занятия |
| Описание | Пожелания по дням недели, времени и техническому оснащению |

## Как работает скрипт

### Шаг 1 — Чтение файлов
Скрипт читает оба CSV-файла с помощью библиотеки pandas.
Разделитель указывается явно (`;`), чтобы корректно распознать все колонки.

### Шаг 2 — Добавление колонки «Семестр»
В каждый файл добавляется служебная колонка `Семестр`:
- строки из `autumn.csv` получают значение `Осень`
- строки из `spring.csv` получают значение `Весна`

Это позволяет в сводном файле всегда понимать, из какого семестра пришла каждая запись.

### Шаг 3 — Объединение данных
Оба файла объединяются в один с помощью `pd.concat`.
Сначала идут все строки из осеннего семестра, затем — из весеннего.
Индексы строк сбрасываются, чтобы нумерация была сквозной.

### Шаг 4 — Сохранение результата
Сводный файл сохраняется в формате CSV с разделителем `;`.
Имя файла формируется автоматически на основе текущего года: `Сводный_2026.csv`.
Кодировка `utf-8-sig` обеспечивает корректное отображение кириллицы при открытии в Excel.

## Результат
Создаётся файл `Сводный_2026.csv` в той же папке.

Параметр | Значение

Строк из весеннего семестра | 645
Итого строк в сводном файле | 1303
Количество колонок | 12
